Modo de uso - HTML

    Modo de uso

  • 1. Sobre o básico/padrão:
  •   Trata-se de um elemento html de uso comum (button) com características diferenciadas.

      Ao utilizar a classe mkBot no elemento button e contiver a classe "atualizar" ou "atualizarSemEvento", a biblioteca irá modificar o conteúdo do botão <button class="mkBot atualizar">CONTEUDO</button> pelo conteúdo do data-value. Mas o data-value só serve para valores iniciais, similar ao placeholder de um elemento input.

      Se houver value (não é nativo do button) no elemento, o botão verificará se este conteúdo for um PDF ou IMAGEM, para então exibir estes dentro do botão, mas quando o usuário clica no botão, não irá clicar no conteúdo do pdf ou imagem, desta forma o gatilho onClick do permanece funcionando mesmo com um pdf dentro do botão. Note que se pdf tiver botões dentro, por padrão estará clicando no botão externo e não nos botões internos.

      Ao clicar executa-se o onclick e ao receber a classe "atualizar", o elemento executará onchange (não é nativo do button) do botão e fará a verificação do conteúdo do value para o algorítmo de exibição do PDF.
    Em função Javascript

      Pré-Requisitos:
    » A biblioteca precisa estar iniciada.

  • 2. Liberar clicar internamente
  •   Com dito a cima, por padrão o botão não bloqueia cliques dentro do objeto interno. Para liberar o usuário de clicar no que estiver dentro do botão utilize data-clicavel="false", o botão perderá prioridade na parte onde o objeto foi renderizado.
    data-clicavel="false"

  • 3. Exibir Barra
  •   Normalmente os navegadores já incrementam alguns objetos, como PDF com uma barra para download e impressão deste pdf por alí mesmo. Quando liberado para clicar internamente, essa barra irá surgir em larguras acima de aproximadamente 250 píxels.

      Para impedir a exibição da imagem, você pode utilizar atributo data-exibirbarra="false".
    data-exibirbarra="false"
