    Modo de uso

  • 1. Sobre o básico/padrão:
  •   Trata-se de um elemento html de uso comum (button) com características diferenciadas.

      Ao utilizar a classe mkBot no elemento button, a biblioteca irá modificar o conteúdo do botão ≶button>CONTEUDO≶/button> pelo valor inicial do argumento do botão data-value.

      Se houver value (não é nativo do button) no elemento, o botão processará um algorítmo de verificação deste conteúdo e se ele for um PDF ou IMAGEM, ele colocará estes dentro do botão, de forma que o usuário consiga clicar no botão sem selecionar o conteudo do pdf ou imagem.

      Ao clicar executa-se o onclick e ao receber a classe "atualizar", o elemento executará onchange (não é nativo do button) do botão e fará a verificação do conteúdo do value para o algorítmo de exibição do PDF.
    Em função Javascript
