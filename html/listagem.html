  • 1. Sobre o básico/padrão:
  •   A Listagem é uma grande ferramenta que contém várias funções interligadas para exibir conteúdos de forma organizada, permitindo ao usuário diversas possibilidades de iteração.
      Pré-Requisitos:
    » A biblioteca precisa estar iniciada.
    » Uma URL de API para trazer dados em formato JSON.
    » Uma tabela com um container que cerca o resumo.
    » Um modelo de linha da tabela (template).
