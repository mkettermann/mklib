  • 1. Sobre o básico/padrão:
  •   Esta é uma ferramenta que utiliza um campo INPUT como no exemplo ao lado. Esse input inicial / principal se transforma em uma ferramenta de seleção assim que a biblioteca for carregada e encontrar esse input no DOM.

      Quando encontrar esse input principal for encontrado, cria-se um container e transfere o style do input principal para este container. Dentro do container se forma o seletor que o usuário interage.
      O seletor todo (container) é composto pelo input inicial, um input de pesquisa onde o usuário mantem o foco para visualizar as opções, e uma lista de opções que se baseia em um atributo do input principal: (Array KV).
      Quando o seletor se forma, a lista de opções é construida baseada Array KV. Caso essa lista for alterada dinamicamente, o seletor não irá atualizar sozinho a lista de opções ao menos que o input inicial receba uma classe "atualizar".
      O filtro é um input simples que fica visível para o usuário. Quando o usuário preenche conteúdo neste campo, o seletor oculta todas as opções que não contém este dado preenchido. Quando não está focado, exibe a quantidade de itens selecionados quando foi liberado selecionar mais de um item ou exibe o item selecionado caso só seja possível selecionar um.
      A lista de opção visível é baseada neste V do array KV. Mudar o V não interfere no item selecionado, pois o V age como a representação visual de K. Se esse atributo não é informado, o sistema tema coletar do atributo refill="url" e se este também não existir, o seletor ficará vazio de opções.
    <input class='mkSel'/>

      Pré-Requisitos:
    » Precisa-se de um INPUT com a classe "mkSel".
    » A biblioteca precisa estar iniciada.

  • 2. Limitador de seleção: (Padrão: 1)
    1. (1) Permite apenas um clique e se fecha (normal).
    2. (-1) Libera até 4 bilhões de itens selecionados.
    3. (2 ou +) Seleciona e permanece aberto (multiselect).
    1. » selapenas="(limite)"
    2. » selapenas="1"
    3. » selapenas="-1"
    4. » selapenas="3"

  • 3. Flexas de Subir e Descer: (Padrão: true)
  •   Quando o seletor possui muitas opções. Automaticamente surgem flexas para subir ou descer. Caso não deseja renderizar elas junto, utilize essa propriedade juntamente com o input.
    data-selmove="false"

  • 4. Array KV (Chave-valor): (Padrão: sem opções visíveis)
  •   A array de opções Chave-Valor representa uma string em formato JSON que o seletor converte para objeto quando necessário. Para popular o seletor com opções para seleção, basta informar o parametro data-selarray e dentro, uma array de objetos com propriedade 'k' e 'v' que, respectivamente, são chave e valor. Como neste exemplo:
    <input class='mkSel atualizar'
  data-selarray='[{"k":"","v":"Todos"},
	{"k":"1","v":"Sim"},{"k":"2","v":"Não"}]' />

      Todos representa vazio, 1 para Sim e 2 para Não. Você pode utilizar JSON para converter ou desconverter nessa propriedade.
      Você pode alterar o conteúdo desta array em formato string. Mas para o seletor atualizar os valores, você precisará adicionar ao input principal a classe: "atualizar", igual ao exemplo acima e o "atualizar" sumirá sozinho se for detectado. Ao atualizar o seletor executará evento de 'input' e 'change'.
      Caso você modifique o seletor após ele já ter sido renderizado, mas não deseja que lance eventos neste input, adicione a classe "atualizarSemEvento" em vez da classe "atualizar". O efeito é o mesmo, mas pode evitar alguma cascata de eventos pré estabelecidos por você no seu projeto.

    2. » data-selarray=""

    5. Classes de atualização
    6. » class="atualizar"
    7. » class="atualizarSemEvento"

  • 5. Personalizar CSS:
  •   Para personalizar o seletor, você pode mudar as variáveis CSS do container. Veja como este código abaixo foi renderizado ao lado: 
    <input class="mkSel"
  style="
    --FundoItensSelArrow: #ffff00
    --FundoItensHover: #00ff00;
    --FundoItensMarK: #ff000055;
    --FundoItensSel: #0000ff55;
    --ArrowSelecionado: '\2764'
    width: 150px;"
  data-selarray='[{"k":"","v":"Todos"},
	{"k":"1","v":"Sim"},{"k":"2","v":"Não"}]' />

  • 5. Modo Desenvolvimento: (Padrão: false)
  •   É uma propriedade que deixa o seletor em um formato aberto para edição e visualização em tempo integral. É possível visualizar um campo contendo os valores chave selecionados, um campo de display e filtro e por fim a lista de opções permanece aberta mesmo sem foco no filtro.
    data-dev="true"

  • 6. Setas Laterais: (Padrão: false)
  •   Permite ao usuário mover o atual item selecionado para cima ou para baixo clicando nas setas. Apenas funciona quando for apenas um selecionavel (selapenas="1"). Para ativar, utiliza-se o data-selmovesel="true" e as flexas em svg surgirão ao lado do campo de filtro e reduzirão o tamanho do filtro.
    data-selmovesel="true"

  • 7. Uso por teclado:
  •   Os seletores possuem, por padrão, a funcionalidade de utilizar as teclas: enter, esc, tab, setas para cima e seta para baixo. Com isso é possível navegar entre os seletores, navegar entre as opções dos seletores, selecionar uma opção ou cancelar apenas utilizando um teclado.
      O Seletor utiliza o atributo "data-m" nos itens para saber em qual linha está o ponteiro. E para os itens selecionados, utiliza o "data-s".
      Por Padrão, o seletor tenta mover o scroll do seletor até a posição do selecionado quando o seletor receber foco e quando utiliza as teclas para cima e para baixo.
    1. » Esc
    2. » Enter
    3. » Seta para Cima
    4. » Seta para Baixo

  • 8. GET e SET das Opções e do Selecionado atual:
  •   Para modificar ou coletar os as opções atuais ou a(s) seleção(ões) atual(is), é possivel utilizando esses comandos em javascript. Como já explicado no item 4, você pode setar diretamente no campo e colocar a classe "atualizar", mas aqui vamos fazer apenas pelo javascript.
      Vamos começar pela parte das opções que o usuário tem. A Array de opções fica no atributo do campo e para isso você precisará do elemento. Para você coletar o que está lá no atual momento, há duas funcões, uma formatos KV e a outra no Map: 
    let meuMap = mkt.mkSelArrayGetMap(document.querySelector(".mkSel"));

      E para modificar as oções no elemento: 
    mkt.mkSelArraySetMap(elemento, meuMap)

      Quando modificar as opções ou as seleções, cria a possibilidade do selecionado não estar na lista de opçoes.
      Se deseja que o campo execute o evento, adicione a classe "atualizar" ao elemento.
