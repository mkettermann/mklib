Seletor


  • Funcionalidade do Seletor
    Modo de uso
  • LIMITADOR DE SELEÇÃO.
    (1) Permite apenas um clique e se fecha (normal).
    (-1) Libera até 4 bilhões de itens selecionados.
    (2 ou +) Seleciona e permanece aberto (multiselect).
    1. » selapenas="(limite)"
    2. » selapenas="1"
    3. » selapenas="-1"
    4. » selapenas="3"
  • O Seletor vem juntamente com a biblioteca MK
    Padrão
  • FLEXAS SOBE E DESCE. Quando o seletor possui muitas opções. Automaticamente surgem flexas para subir ou descer. Caso não deseja renderizar elas junto, utilize essa propriedade juntamente com o input.
    data-selmove="false"
  • FILTRO. O Seletor possui filtro sob listagem de opções com reação direta por tecla pressionada.
    Padrão
  • ARRAY DE OPÇÕES Chave-Valor. Para popular o seletor com opções para seleção, basta informar o parametro data-selarray e dentro, uma array de objetos com propriedade 'k' e 'v' que, respectivamente, são chave e valor. Neste exemplo: [{"k":"","v":"Todos"},{"k":"1","v":"Sim"},{"k":"2","v":"Não"}] Todos representa vazio, 1 para Sim e 2 para não. Você pode utilizar JSON para converter ou desconverter nessa propriedade.
    data-selarray=""
  • MODO DEV é uma propriedade que deixa o seletor em um formato aberto para edição e visualização em tempo integral. É possível visualizar um campo contendo os valores chave selecionados, um campo de display e filtro e por fim a lista de opções permanece aberta mesmo sem foco no filtro.
    data-dev="true"
  • SETAS LATERAIS. Move o item selecionado (1 apenas) para cima ou para baixo clicando nas setas.
    data-selmovesel="true"
  • USO POR TECLADO. Os seletores possuem a funcionalidade de utilizar as teclas: enter, esc, tab, setas para cima e seta para baixo. Com isso é possível navegar entre os seletores, navegar entre as opções dos seletores, selecionar uma opção ou cancelar apenas utilizando um teclado. O Seletor utiliza o atributo data-m nos itens para saber em qual linha está o ponteiro. E para os itens selecionados, utiliza o data-s.
    Por Padrão, o seletor tenta mover o scroll do seletor até a posição do selecionado quando o seletor receber foco e quando utiliza as teclas para cima e para baixo.
    1. » Esc
    2. » Enter
    3. » Seta para Cima
    4. » Seta para Baixo
