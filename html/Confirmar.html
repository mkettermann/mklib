    Modo de uso

  • 1. Sobre o básico/padrão:
  •   Trata-se de uma função que gera um diálogo de confirmação com o usuário, solicitando uma decisão do tipo Sim ou Não apenas.

      A função mkConfirma() Usa-se em javascript, dentro da função que precisa de confirmação. O parâmetro de mkConfirma é uma string contendo a pergunta do corpo do diálogo. Por se tratar de uma função assíncrona, o sistema aguardará o usuário clicar no botão.

      Quando o usuário decidir sim/não, o sistema irá retornar a função dentro do .then() contendo true para sim ou false para não. Também é possível usar o async/await e colocar a resposta em uma variável. Como no exemplo acima. Foi feito uma condicional para saber se o usuário respondeu true (Sim) ou false (Não).
