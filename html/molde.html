  • 1. Sobre o básico/padrão:
  •   Molde se trata de uma ferramenta de clonagem de estrutura.
      Primeiramente, precisa definir um template / modelo que representa a estrutura que os dados serão colocados. No exemplo acima, foi definido o modelo do lado esquerdo da tabela e setado um id para esta tag html com nome "modelo". É possível utilizar id/class, informando da mesma forma que se referencia ele no CSS ('#' para id, '.' para classe).
      Dentro do modelo você utiliza os caracteres '${' e '}' para iniciar e finalizar o acesso a propriedade do objeto. Não pode haver espaços ou outros caracteres além dos caracteres exatos e na mesma ordem da propriedade do objeto.
      No exemplo acima, o modelo a esquerda contem ${nome}, então será procura no objeto a propriedade "nome". Na coluna do centro há um input que contem um atributo name='nome', assim esse input receberá um valor para a propriedade "nome". Há também uma função 'aoModificarInput' na página que executa esse input quando iniciar a página ou modificar o campo.
      A função que executa o Molde também é, neste caso, a mesma função que seta o nome da propriedade no objeto. Veja a função: 
    	function aoModificarInput(e) {
		objeto[e.name] = e.value;
		mk.mkMoldeOA(objeto, "#modelo", ".resultado");
	}
